## What We Did

- **Startup performance overhaul** — fixed `window is not defined` crash by changing `app.json` output from `"static"` to `"single"`; rewrote `lib/supabase.ts` to lazy-init the Supabase client via `Proxy` (defers `createClient()` until first use, fixes 5+ second cold start); removed blocking `getSession()` call from auth store; replaced with `INITIAL_SESSION` event + JWT `user_metadata` (no DB round-trip on startup; cold start now <1 second)
- **Fixed navigator crash** — replaced `useEffect + router.replace()` in `app/index.tsx` with `<Redirect>` component; fixes "Attempted to navigate before mounting Root Layout"
- **Created `lib/debug.ts`** — timestamped logger with `APP_START` baseline for tracing startup sequence
- **PGRST116 guard** — patient dashboard now shows empty state instead of error screen when no patient record is linked to auth user
- **Logout buttons** — added to clinic home header and patient dashboard header (all render states)
- **Supabase backend wiring** (committed prior to session compaction):
  - Assessment "Save to Cloud" inserts to `classification_results`, updates session status (GAP-07)
  - History screen loads real sessions by `clinic_id` (GAP-09)
  - Admin users screen reads `profiles` table; Activate/Deactivate wired (GAP-10, UX-08)
  - Admin clinics screen reads `clinics` + `devices`; Activate/Deactivate wired (GAP-11)
  - Both session detail screens load real Supabase data with joins (UX-07)
  - Clinic dashboard: real clinic name + today's stats from Supabase (CODE-11)
  - Live-feed foot selector `onPress` wired; active style mirrors state (BUG-05)
  - Admin overview stats wired to Supabase counts (S-01)
  - Admin clinic cards navigate to clinics tab (N-02)
- **Icon standardization (UX-14)** — comprehensive sweep of all emoji and unclear Unicode symbols across 20 files; replaced with `@expo/vector-icons` Ionicons:
  - Tab bar `_layout.tsx` — `TabIcon` component rewritten with typed Ionicons prop
  - Auth screens — `◈` brand logo → `pulse-outline`
  - All `›` chevrons → `chevron-forward`
  - All `←` back arrows → `arrow-back-outline`
  - All `⚠`/`✓` inline symbols → `warning-outline`/`checkmark-circle-outline`
  - All `📡 📋 🧠` empty-state / processing icons → appropriate Ionicons
  - All `→ ↑` arrow suffixes removed from button labels
