## What We Did

### Crash Fix
- Diagnosed and fixed "Cannot assign to read-only property 'NONE'" crash
- Root cause: `babel.config.js` had `@babel/plugin-proposal-decorators` listed twice — once manually and once via `babel-preset-expo` which already includes it. Duplicate execution corrupted class property assignments.
- Fix: removed both manual plugin entries; `babel-preset-expo` handles everything
